URL Shortener Microservice

NOTE: Shortened URLS are valid for 24hrs

Example Usage:

To create a shortened URL:
  • Submit a valid URL using the input form above.
  • Alternatively, submit a POST request to /api/shorturl with a body containing a url encoded field of 'url' (e.g. 'url=https://google.com')
To use a Shortened URL:
  • Visit /api/shorturl/<SHORTURL>, with a valid SHORTURL, to be redirected to the original URL.
    A shareable link can be found under the "short_link" property of the returned JSON on submitting a URL to be shortened.
    For example, this short url will redirect you to google.com:
    fcc-backend-project3.plcoster.repl.co/api/shorturl/G00Gl3

Example Output:

On submitting a URL to be shortened, a JSON of the following form will be returned:

{
"original_url": "https://google.com",
"short_url": "G00Gl3",
"short_link":"https://fcc-backend-project3.plcoster.repl.co/api/shorturl/G00Gl3"
}
